# 00 · 总览与直觉核对

## 0.1 你给出的直觉（按原话整理）

1. BLIP2 看到相似物体，把"去掉障碍后的锥形视角"投到 2D。
2. **通道 1 的 current value** 由相似度决定，**通道 2 的 current confidence** 由"锥形里靠近光轴"决定。
3. 当前帧 2D 投射如果与历史 **没有重合**，那块 3D 像素直接写入双通道；
4. 如果 **有重合**，按公式用 $v_{prev}, v_{curr}, c_{prev}, c_{curr}$ 算 $v_{new}, c_{new}$。
5. YOLOv7（识别）、GroundingDINO（切割）、BLIP2 三者同时跑；YOLO / Grounding **层级高于** BLIP2，"YOLO 有结果就覆盖 BLIP 给出的下一个走向点"，目标点是走向**最近点**。

## 0.2 一句话评分

> **大方向对，但有 3 个关键细节需要修正。**

修正项预览（详细推导见 01～04）：

| 直觉 | 实际代码 | 结论 |
| --- | --- | --- |
| 通道 1=value, 通道 2=confidence | 二者**不是 value_map 的两个通道**，而是 **两张独立的二维图**：`self._value_map (H,W,C)` 与 `self._map (H,W)`。`C` 取决于 prompt 个数（V2=1，V3=2，两通道都是 BLIP2 余弦）。 | ❗ 修正 |
| YOLO/Grounding"覆盖" BLIP2 | 不是同一目标的两路投票。YOLO/Grounding 输出走的是 **ObjectPointCloudMap**（决定 "navigate" 模式下的目标 xy）；BLIP2 余弦走的是 **ValueMap**（决定 "explore" 模式下选哪个 frontier）。覆盖关系发生在 `act()` 的 if-else：检测到 → navigate；没检测到 → explore。 | ❗ 修正 |
| "走向最近点" | 是 DBSCAN 滤掉噪声后取离机器人**当前位置最近的点云点**（≥1m 的最近点，<1m 视为太近不可信）。 | ✅ 基本对 |
| 锥形 = depth 截断后的可视扇形 | 对。`_process_local_data` 用 `cone_mask` 减去深度阻挡形成 `visible_mask`。 | ✅ |
| 靠近光轴 confidence 高 | 对。`cos²(θ)` 后被 remap 到 `[0.25, 1]`。 | ✅ |
| 重合区按公式融合 v 和 c | 对。**有两种模式**：max-confidence 或加权平均（论文公式）。代码里两种都实现，由 `use_max_confidence` 切换。**HM3D/MP3D/Spot 实际跑的 yaml 默认是 `False`，即加权平均。** | ✅ 但有两种实现 |

## 0.3 修正后的"标准直觉"

```
RGB(H,W,3) ─► YOLOv7 / GroundingDINO ─► detections (bbox, class, conf)
                        │
                        ├─► 命中目标？
                        │     ├─是→ SAM 抠 mask → 点云 → ObjectPointCloudMap
                        │     │   └─► act() 进入 navigate 模式，goal = 最近点
                        │     └─否→ 继续 explore
                        │
RGB(H,W,3) ─► BLIP2-ITM 余弦(每个 prompt 一个标量)
                        │
                        ▼
Depth(H,W) + tf + fov ─► ValueMap.update_map
                        │
                        ├─► _process_local_data: 锥形 mask − 障碍 = visible_mask (HxW float ∈ [0,1])
                        │     里面的值 = cos²(光轴夹角) ∈ [0.25, 1]   ← 这是 "current confidence" c_curr
                        │
                        ├─► _localize_new_data: 旋转到 episodic frame，贴到 size×size 大图
                        │
                        └─► _fuse_new_data:
                              软门控: 若 c_curr<0.35 且 c_curr<c_prev → c_curr=0 (silence)
                              if use_max_conf (ValueMap 类内默认=True, 但 HM3D/MP3D/Spot yaml 设为 False):
                                  谁 c 大用谁，含"没历史→直接写"
                              else (HM3D 实际默认, 论文公式):
                                  w1 = c_prev/(c_prev+c_curr), w2 = c_curr/(c_prev+c_curr)
                                  v_new[k] = w1·v_prev[k] + w2·v_curr[k]
                                  c_new    = w1·c_prev + w2·c_curr = (c_prev² + c_curr²)/(c_prev+c_curr)
                                  (nan_to_num 处理 c_prev=c_curr=0 的像素)

ObstacleMap.update_map(depth)        # 高度过滤的 obstacle，加 explored area
       │
       ▼
ObstacleMap.frontiers (N,2)   ◄── 这是 explore 模式下可选的下一目标
       │
       ▼
ValueMap.sort_waypoints(frontiers) → 取 value 最大的 frontier
       │
       ▼
WrappedPointNavResNetPolicy.act(rho, theta, nav_depth) → 离散动作 / Spot 的角速度+线速度
```

## 0.4 三个模型实际运行频率

| 模型 | 何时调用 | 走什么地图 |
| --- | --- | --- |
| **YOLOv7** | 每一步、且 target 在 COCO 类里时（`_load_yolo=True`） | ObjectPointCloudMap |
| **GroundingDINO** | 每一步、target 不在 COCO，或 YOLO 漏检后兜底 | ObjectPointCloudMap |
| **MobileSAM** | 仅在 YOLO/Grounding 出 bbox 时，对每个 bbox 抠 mask | ObjectPointCloudMap |
| **BLIP2-ITM** | 每一步（V2/V3 总是；ITMPolicy 仅可视化时） | ValueMap |
| **BLIP2 (VQA)** | 可选，`use_vqa=True` 时对 SAM 的 mask 做 "Is this a chair?" 确认 | ObjectPointCloudMap 写入前 |

所以 BLIP2-ITM 和 YOLO/Grounding **确实是并行**（针对同一帧 RGB 都跑一遍），只是产物落到 **不同地图**，不存在"覆盖 BLIP2 的下一个走向点"——而是 `act()` 的优先级：

```python
# vlfm/policy/base_objectnav_policy.py: BaseObjectNavPolicy.act
if not self._done_initializing:        # 第 0~10 步原地转身建图
    pointnav_action = self._initialize()
elif goal is None:                     # ObjectPointCloudMap 没目标
    pointnav_action = self._explore(observations)   # ← 用 ValueMap 选 frontier
else:                                  # 检测到目标
    pointnav_action = self._pointnav(goal[:2], stop=True)  # ← 直奔最近点
```
