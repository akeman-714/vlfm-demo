# 05 · 中间数据形态速查表

> 整理本仓库一次 `act()` 涉及到的所有"中间产物"。单位、形状、取值范围、来源代码行。
>
> 默认场景：HM3D ObjectNav，Habitat，RGB-D 640×480，HFOV 79°，min/max_depth=0.5/5.0m。

## 5.1 传感器与观测

| 名称 | 来源 | shape | dtype | 单位 / 范围 |
| --- | --- | --- | --- | --- |
| `observations["rgb"]` | Habitat sim | (1, 480, 640, 3) | uint8 | 0–255 |
| `observations["depth"]` | Habitat sim | (1, 480, 640, 1) | float32 | 0–1（已被 sim 归一化） |
| `observations["gps"]` | gps_sensor | (1, 2) | float32 | 米 (x, -y) |
| `observations["compass"]` | compass_sensor | (1, 1) | float32 | 弧度 yaw |
| `observations["heading"]` | heading_sensor | (1, 1) | float32 | 弧度 |
| `observations["objectgoal"]` | ObjectGoalSensor | (1, 1) | int | 0~5 (HM3D) / 0~20 (MP3D) |
| `observations["frontier_sensor"]` | frontier_sensor lab_sensor | (N, 2) | float32 | 米；当 `compute_frontiers=True` 时来自 ObstacleMap |
| 滤波后 depth | `depth_camera_filtering.filter_depth` | (480, 640) | float32 | 0–1 |

## 5.2 缓存 `_observations_cache`

`habitat_policies.py:210-237`

```python
{
  "frontier_sensor": np.ndarray (N, 2) float,    # 米
  "nav_depth":       Tensor (1, 480, 640, 1),    # 给 pointnav
  "robot_xy":        np.ndarray (2,) float,      # 米
  "robot_heading":   float,                      # 弧度
  "object_map_rgbd": [(rgb(480,640,3), depth(480,640), tf(4,4), 0.5, 5.0, fx≈391, fy≈391)],
  "value_map_rgbd":  [(rgb(480,640,3), depth(480,640), tf(4,4), 0.5, 5.0, 1.379)],
  "habitat_start_yaw": float,
}
```

`fx = fy = image_width / (2·tan(hfov/2)) = 640 / (2·tan(79°/2)) ≈ 390.6`
